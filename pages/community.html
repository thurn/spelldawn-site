To stay up to date with the project, take a look at the following resources:

  • https://www.reddit.com/r/spelldawn/ is the Spelldawn subreddit. This is a good place for any broader discussion about the game.
  • https://github.com/thurn/spelldawn/issues is the issue tracker for the project. This is a good place to find open items to work on or submit bugs you encounter.
  • https://github.com/thurn/spelldawn-site contains the code for the website you are currently reading. This site functions like a wiki, feel free to submit a pull request to fix any issues you see or add entirely new useful pages!
