--- title: Contributing Code layout: page ---

If you are interested in technically contributing to the Spelldawn project, the subpages here give information about the architecture of the game and different ways in which you can participate. Contributing Code - Getting Started describes how to set up a development environment and submit code to the project.

https://github.com/thurn/spelldawn/issues is the issue tracker for the project. This is a good place to find open items to work on or submit bugs you encounter.
