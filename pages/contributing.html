If you are interested in contributing to the Spelldawn project, the subpages below give information about different ways in which you can participate.

  • Contributing - Code describes how to set up a development environment and submit code to the project.
  • Contributing - Game Design describes how to help out by submitting card designs or participating in other game design activities.
  • Contributing - Art talks about how to get involved in creating art for the project.
